あなたの Web サイト。

iWeb、Pages など、Mac OS X で動作するさまざまなアプリケーションを使えば、
自分の Web サイトを短時間で簡単に作成および公開できます。

Mac なら、あっという間に自分の Web サイトを作成して公開できます。サイトの準備ができたら、それを公開するのはとても簡単です。

“システム環境設定”を開き、“共有”をクリックして、“Web 共有”を選択してください。

これで完了。あなたのサイトが、自宅や職場のプライベートネットワークで見られるようになりました。

インターネットに接続しているなら、世界中の友人にも自分の Web サイトを見てもらえます。“共有”環境設定に表示されるアドレスを友人たちに教えてあげましょう。

Apache のパワー

Web 共有は、Apache Web サーバ上に構築されます。この業界標準テクノロジーは Mac OS X に含まれています。Apache Web サーバについて詳しくは、Apache のマニュアルを参照してください。
